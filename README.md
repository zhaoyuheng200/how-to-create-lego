# How to Create LEGO

## MicroCAD Rust Tutorial

This is a tutorial for building a micro Computer-Aided Design (CAD) application in Rust. The goal is to create a simple CAD system that can be used to design and model LEGO pieces.

## Getting Started

### Prerequisites

- Rust and Cargo (install from [rustup.rs](https://rustup.rs/))

### Building

```bash
cargo build
```

### Running

```bash
cargo run
```

## Project Structure

- `src/main.rs` - Main entry point for the application
